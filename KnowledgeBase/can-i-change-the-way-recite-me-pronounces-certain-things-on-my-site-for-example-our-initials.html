Can I change the way Recite Me pronounces certain things on my site - for example our initials

Created: 2026-06-16T14:33:21.498532+00:00
Question: Can I change the way Recite Me pronounces certain things on my site - for example our initials?

Answer

Non-technical explanation

Yes. You can override how specific words or acronyms are spoken by defining a pronunciation map in toolbar options.

For initials like USU, add a mapping so text-to-speech reads it the way you want.

Technical explanation

Set your pronunciation overrides in Dom.word_filters.

Important: keys are matched case-sensitively in many setups, so use lowercase keys consistently and test with the exact text your page outputs.

Example mapping:

// Add this to your Recite options configuration.
const options = {
  Dom: {
    word_filters: {
      usu: "you-ess-you"
    }
  }
};

JavaScript example

// Verify your settings at runtime in browser devtools:
Recite.getOptions("Dom");

// Expected shape:
// { word_filters: { usu: "you-ess-you" } }

If word_filters is missing, your options are not being applied at the right point in page load.

After confirming the option is present, test TTS directly on content containing USU.
