How do i add the toolbar to my website

Created: 2026-06-16T13:49:36.052913+00:00
Question: How do i add the toolbar to my website?

Answer

Non-technical explanation

To add the Recite Me toolbar to your website, you need to include a single script tag in your HTML. This script will automatically display a toolbar launcher button (with icon and text) in the bottom-right corner of your page. You’ll need to replace YOUR_CLIENT_ID with your actual Recite Me client ID. The toolbar will load only when the user interacts with the button or meets specific trigger conditions (e.g., URL parameters).

---

Technical explanation

  1. Add the script tag to your website’s HTML. Replace YOUR_CLIENT_ID with your Recite Me client ID:
   <script
     src="https://cdn.reciteme.com/reciteme_toolbar_launcher.min.js"
     data-clientid="YOUR_CLIENT_ID">
   </script>

This script loads the toolbar launcher and defaults to rendering a button with the style icon-text and position floating-bottom-right [1].

  1. Required attribute:
  • data-clientid: Your Recite Me client ID (mandatory for functionality) [1].
  1. Default behavior:
  • A single toolbar launcher button is created.
  • The button’s appearance and position follow default settings (see table below) [1].
  1. Optional customizations:
  • Change the button’s position using data-button-toolbar-position (e.g., floating-top-left, inline).
  • Adjust colors, icons, or text via attributes like data-button-toolbar-background-color or data-button-toolbar-text [5].

---

Recommended next steps

  1. Verify your client ID: Ensure YOUR_CLIENT_ID is correct and matches your Recite Me account [3].
  2. Test mobile behavior: Check that the toolbar’s position and styling adapt correctly for mobile devices using -mobile overrides [4].
  3. Validate accessibility: Confirm the button’s contrast and focus visibility meet accessibility standards [3].
  4. Test triggers: Ensure the toolbar loads properly when triggered (e.g., button click, URL parameters) [2].
